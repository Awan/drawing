#### latest NODE installation in debian

```
$ echo 'deb https://deb.nodesource.com/node_12.x sid main' | sudo tee -a /etc/apt/sources.list > /dev/null

$ sudo apt update
$ sudo apt install -y nodejs 
```

Enjoy using NODEjs!
